    About

    The minecraft jigsaw preview tool (the "Tool") can display generated jigsaw structures in minecraft. Both vanilla structures and custom structures from datapacks can be displayed. The tool is developed, maintained and made availabe by Jochen Jacobs (the "Author"). The source code of the Tool is avalable at https://github.com/jacobsjo/minecraft-jigsaw-preview/ under the MIT licence.

    Privacy Policy

    The Tool does not collect any user data and does not use any cookies. Data selected using the "Open Datapack .zip" and "Open Datapack Folder" buttons are not transfered to a server and only read on the local machine of the user. In some cases, the Tool may read changes to files inside the datapack folder done after the inital selection.

    The Tool is hosted using GitHub pages. GitHub may collect user personal information from visitors of the tool, including logs of visitor IP addresses, to comply with legal obligations, and to maintain the security and integrity of the Website and their Service. You can find the privacy policy of GitHub here. No data collected by GitHub is accessible by the Author or any third party.
